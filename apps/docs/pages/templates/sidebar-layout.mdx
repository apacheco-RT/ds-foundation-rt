# SidebarLayout

Fixed-width sidebar with a scrollable main content area. Use this when the sidebar width is known and constant ‚Äî app nav, filter panels, detail drawers.

## Usage

```tsx
import { SidebarLayout } from '@ds-foundation/react';

<SidebarLayout sidebar={<AppNav />}>
  <div className="p-6">
    Main content
  </div>
</SidebarLayout>
```

Custom sidebar width:

```tsx
<SidebarLayout sidebar={<FilterPanel />} sidebarWidth="20rem">
  <ResultsList />
</SidebarLayout>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `sidebar` | `ReactNode` | ‚Äî | **Required.** Sidebar content |
| `sidebarWidth` | `string` | `"16rem"` | CSS width value for the sidebar (e.g. `"20rem"`, `"240px"`) |
| `children` | `ReactNode` | ‚Äî | **Required.** Main content |
| `className` | `string` | ‚Äî | Additional classes on the root wrapper |
| `...rest` | `HTMLAttributes<HTMLDivElement>` | ‚Äî | Forwarded to the root `<div>` |

## Notes

- Root element is `flex h-full` ‚Äî inherits height from its container
- Both sidebar and main scroll independently
- For responsive behaviour (collapsing sidebar on mobile), wrap in your own media query logic
