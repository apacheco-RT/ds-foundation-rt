# PageLayout

Full-page shell with optional header, sidebar, footer, and a scrollable main content area. All slot props are optional вҖ” omit any you don't need.

## Usage

```tsx
import { PageLayout } from '@ds-foundation/react';

<PageLayout
  header={<AppHeader />}
  sidebar={<AppNav />}
  footer={<AppFooter />}
>
  <div className="p-6">
    Page content here
  </div>
</PageLayout>
```

Without sidebar or footer:

```tsx
<PageLayout header={<AppHeader />}>
  <main className="p-6">
    Simple page with just a header
  </main>
</PageLayout>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `header` | `ReactNode` | вҖ” | Rendered in `<header>` above the content area |
| `sidebar` | `ReactNode` | вҖ” | Rendered in `<aside>` to the left of main |
| `footer` | `ReactNode` | вҖ” | Rendered in `<footer>` below the content area |
| `children` | `ReactNode` | вҖ” | **Required.** Main content area |
| `className` | `string` | вҖ” | Additional classes on the root wrapper |
| `...rest` | `HTMLAttributes<HTMLDivElement>` | вҖ” | Forwarded to the root `<div>` |

## Notes

- Root element is `min-h-screen flex flex-col` вҖ” fills the viewport by default
- Sidebar slot has `flex-shrink-0`; size it with a width class or inline style on the child
- Main area uses `overflow-auto` вҖ” scrolls independently when content exceeds available height
