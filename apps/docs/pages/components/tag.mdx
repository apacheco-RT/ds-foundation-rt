# Tag

Pill-shaped label with semantic colour variants. Supports optional icons and a remove button.

## Usage

```tsx
import { Tag } from '@ds-foundation/react';

<Tag>Default</Tag>
<Tag variant="blue">In Progress</Tag>
<Tag variant="green">Complete</Tag>
<Tag variant="error">Failed</Tag>
<Tag variant="orange">Attention</Tag>
<Tag variant="purple">Brand</Tag>
```

With a remove button:

```tsx
<Tag variant="blue" onRemove={() => removeTag('in-progress')}>
  In Progress
</Tag>
```

With an icon:

```tsx
import { Clock } from 'lucide-react';

<Tag variant="orange" icon={<Clock className="w-3 h-3" />}>
  Pending
</Tag>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "blue" \| "green" \| "error" \| "orange" \| "purple"` | `"default"` | Colour intent |
| `icon` | `ReactNode` | — | Icon rendered before the label |
| `onRemove` | `() => void` | — | Renders a close button; called on click |
| `children` | `ReactNode` | — | Tag label content |
| `...rest` | `HTMLAttributes<HTMLSpanElement>` | — | Forwarded to the `<span>` element |

## Accessibility

The remove button has `aria-label="Remove"` for screen reader clarity.
